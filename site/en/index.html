Ecosystem Modules

Discover the repositories that make up the core Numo infrastructure, available on GitHub.

numoCore

The settlement engine. Validates transactions, participants, and balances, executing atomic settlement and ISO 20022 message processing.

GO / POSTGRES / GRPC

numoLedger

The financial ledger. Records all debit and credit entries immutably, ensuring real-time balances.

GO / POSTGRES

numoStream

Event bus that connects all modules asynchronously, guaranteeing high throughput and resilience (Event Sourcing).

KAFKA / AVRO

numoDirectory

Official catalog of local and external participants authorized to operate in the settlement network.

GO / FASTAPI

numoKeys

Centralized key directory. Resolves flexible identifiers (phone, email) with sub-10ms resolution latency.

GO / SCYLLA_DB

numoTrust

Identity management (PKI). Dynamic certificate issuance under mTLS 1.3 and native HSM integration via PKCS#11 and SPIRE.

VAULT / SPIRE

numoAdmin

Administrative backend and API. Provides control routes and acts as a proxy for the operational frontend.

NODE.JS / NESTJS

numoAdmin-next

Modern operational dashboard for payment auditing, participant management, and ecosystem monitoring.

NEXT.JS / REACT
