Módulos do Ecossistema

Conheça os repositórios que compõem a infraestrutura base do Numo, disponíveis no GitHub.

numoCore

O motor principal. Valida transações, participantes e saldos, executando a liquidação atômica e processamento de mensagens ISO 20022.

GO / POSTGRES / GRPC

numoLedger

O livro-razão financeiro. Registra todas as entradas de débito e crédito de forma imutável, garantindo balanço em tempo real.

GO / POSTGRES

numoStream

Barramento de eventos que conecta todos os módulos de forma assíncrona, garantindo alta vazão e resiliência (Event Sourcing).

KAFKA / AVRO

numoDirectory

Catálogo oficial de participantes locais e externos autorizados a operar na rede de liquidação.

GO / FASTAPI

numoKeys

Diretório de endereçamento centralizado. Resolve identificadores flexíveis (telefone, email) com latência inferior a 10ms.

GO / SCYLLA_DB

numoTrust

Gestão de identidades (PKI). Emissão dinâmica de certificados sob mTLS 1.3 e integração nativa com HSM via PKCS#11 e SPIRE.

VAULT / SPIRE

numoAdmin

Backend administrativo e API. Fornece as rotas de controle e age como proxy para o frontend operacional.

NODE.JS / NESTJS

numoAdmin-next

Dashboard operacional moderno para auditoria de pagamentos, gestão de participantes e monitoramento do ecossistema.

NEXT.JS / REACT

Arquitetura Soberana

O Numo atua como a entidade central (Nó Raiz) concebida para Bancos Centrais e Reguladores. Instituições Financeiras (IFs) e de Pagamentos (IPs) conectam-se a este motor seguro para transacionar, eliminando a fragmentação do sistema.

Visão Open Finance

A parte central do ecossistema: atuando como repositório de diretórios, conectividade, monitoramento, governança e indicadores operacionais.

Replicamos os pilares de sucesso de jurisdições avançadas para suportar a inovação em escala nacional.
