Módulos del Ecosistema

Conozca los repositorios que conforman la infraestructura base de Numo, disponibles en GitHub.

numoCore

El motor principal. Valida transacciones, participantes y saldos, ejecutando la liquidación atómica y procesamiento de mensajes ISO 20022.

GO / POSTGRES / GRPC

numoLedger

El libro mayor financiero. Registra todas las entradas de débito y crédito de forma inmutable, garantizando balances en tiempo real.

GO / POSTGRES

numoStream

Bus de eventos que conecta todos los módulos de forma asincrónica, garantizando alto rendimiento y resiliencia (Event Sourcing).

KAFKA / AVRO

numoDirectory

Catálogo oficial de participantes locales y externos autorizados para operar en la red de liquidación.

GO / FASTAPI

numoKeys

Directorio de direccionamiento centralizado. Resuelve identificadores flexibles (teléfono, email) con latencia inferior a 10ms.

GO / SCYLLA_DB

numoTrust

Gestión de identidades (PKI). Emisión dinámica de certificados bajo mTLS 1.3 e integración nativa con HSM vía PKCS#11 y SPIRE.

VAULT / SPIRE

numoAdmin

Backend administrativo y API. Proporciona rutas de control y actúa como proxy para el frontend operativo.

NODE.JS / NESTJS

numoAdmin-next

Tablero operativo moderno para auditoría de pagos, gestión de participantes y monitoreo del ecosistema.

NEXT.JS / REACT

Arquitectura Soberana

Numo actúa como la entidad central (Nodo Raíz) diseñada para Bancos Centrales y Reguladores. Las Instituciones Financieras y de Pagos se conectan a este motor seguro para realizar transacciones, eliminando la fragmentación del sistema nacional.

Mantenido por Ti.

Numo no está gobernado por una fundación privada o corporativa. El proyecto se está estructurando para ser guiado colaborativamente por una comunidad global transparente. Buscamos Especialistas Financieros, Auditores de Seguridad y Desarrolladores C/Rust/Go para auditar el código que moverá a su país.
