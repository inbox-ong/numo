# Numo (repositorio principal)

![Logo Numo](numo.png)

Este repositorio serve como ponto central de documentacao do projeto Numo.
Cada posta/modulo esta em um repositorio separado e este repo agrega contexto,
visao geral e links para cada parte do sistema.

## Estrutura

- `docs/overview.md`: visao geral do projeto
- `docs/architecture.md`: arquitetura e fluxos principais
- `docs/security.md`: seguranca, identidade e mTLS
- `docs/observability.md`: logs, metricas e tracing
- `docs/contracts.md`: contratos e versionamento de schemas
- `docs/testing.md`: estrategia de testes e CI/CD
- `docs/contributing.md`: guia de contribuicao e uso de IA
- `docs/glossary.md`: termos do dominio
- `docs/repositories.md`: catalogo de repositorios (posta/modulo) com links
- `docs/modules/`: detalhes por modulo
- `roadmap.md`: marcos e prioridades
